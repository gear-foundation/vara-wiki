# Networks

Vara.eth runs on Ethereum. The testnet is deployed on **Ethereum Hoodi**, and mainnet is live on **Ethereum Mainnet**.

## Testnet (Ethereum Hoodi)

Vara.eth is currently running on Ethereum Hoodi testnet.

- Chain ID: `560048`
- Primary RPC: `https://hoodi-reth-rpc.gear-tech.io`
- Explorer: [hoodi.etherscan.io](https://hoodi.etherscan.io)

For the full endpoint matrix (HTTPS, WebSocket, faucet, health checks, and connection examples), use:

→ [Network Endpoints](/docs/vara-eth/reference/network-endpoints)

For Router and related contract addresses, use:

→ [Contract Addresses](/docs/vara-eth/reference/contract-addresses)

## Mainnet (Ethereum Mainnet)

Vara.eth mainnet is live on Ethereum Mainnet.

- Chain ID: `1`
- Primary RPC: `https://mainnet-reth-rpc.gear-tech.io`
- Explorer: [etherscan.io](https://etherscan.io)

For the full endpoint matrix (HTTPS, WebSocket, Beacon, Validator RPCs), use:

→ [Network Endpoints](/docs/vara-eth/reference/network-endpoints)

For Router and wVARA contract addresses, use:

→ [Contract Addresses](/docs/vara-eth/reference/contract-addresses)
