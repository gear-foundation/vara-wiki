# Networks

Vara.eth runs on Ethereum. The testnet is deployed on **Ethereum Hoodi** (testnet). Mainnet deployment on Ethereum mainnet is planned.

### Vara.eth RPC (for pre-confirmations)

To send messages and receive instant pre-confirmed results from executors, connect to the Vara.eth HTTPS/WebSocket endpoints:

```
https://vara-eth-validator-1.gear-tech.io
```

```
wss://vara-eth-validator-1.gear-tech.io:9944
```

<Callout type="note" title="Endpoint options">
  More endpoints: `wss://vara-eth-validator-{1..4}.gear-tech.io:9944`
</Callout>

This endpoint is used by the SDK to subscribe to execution results before they're finalized on Ethereum.
