# Developer Hub

This page provides technical resources for developers and integrators working with the Vara â†” Ethereum Bridge.  
It includes the addresses of deployed smart contracts and on-chain programs on both Ethereum and Vara networks for two deployments - Mainnet and Testnet.

:::info Bridge Access

- **Bridge:** [https://bridge.vara.network/](https://bridge.vara.network/)

:::

**Use this section to:**

- Explore and verify bridge-related contracts
- Begin building integrations with the bridge (e.g. relayers, token managers, custom assets)
